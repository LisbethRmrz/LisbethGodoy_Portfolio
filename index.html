PORTAFOLIO


Sistema de Inventario

Este programa fue creado utilizando C# y SQL Server, tiene como finalidad llevar el inventario de productos y ventas de una tienda dedicada a la venta de múltiples productos para el hogar.

Editor de Texto

Este proyecto consiste en un editor online, que le permite a los usuarios personalizar sus textos utilizando los botones, así como ingresar código html para realizarlo; puede ser utilizado en páginas web tipo blog. Está desarrollado haciendo uso de tecnologías como: HTML5, CSS3 y Javascript.

Veterinaria

Proyecto Veterinaria, es un sitio web de control de citas, servicios y productos relacionados a mascotas. Fue desarrollado utilizando HTML5, CSS3, Javascript, Bootstrap, Firebase y Angular.

ParserSQL

ParserSQL, desarrolla las tres primeras fases de un compilador. Lleva a cabo las actividades de un analizador lexico, sintáctico y semántico del lenguaje SQL; está desarrollado haciendo uso de SQL Server y el lenguaje de programación C#.

Sistema de Notas

Sistema de notas, es una aplicación de escritorio para la gestión de información de docentes y estudiantes, así como sus respectivas calificaciones. Está desarrollado haciendo uso de SQL Server y el lenguaje de programación C#.

Adventure Tours

Adventure Tours, es una página web de solicitud y reserva de viajes nacionales o internacionales. Está desarrollado haciendo uso de HTML5, CSS3, Javascript, Bootstrap, MySQL y PHP.
